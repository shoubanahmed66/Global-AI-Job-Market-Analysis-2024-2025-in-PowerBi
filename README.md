
# Global AI Job Market Analysis (2024–2025) in Power BI

## Overview

This project presents a comprehensive analysis of the global AI job market using Power BI and real-world AI job datasets from 2024–2025. The dashboard provides insights into job demand, salary trends, required skills, remote work opportunities, and hiring patterns across different countries and industries.

## Key Features

* Interactive Power BI dashboards with dynamic visualizations
* Analysis of AI job roles, salaries, and experience levels
* Identification of top in-demand AI skills and technologies
* Comparison of remote, hybrid, and onsite job opportunities
* Country-wise and industry-wise hiring trend analysis

## Tools & Technologies

* Power BI
* CSV Dataset
* Data Cleaning & Transformation
* Data Visualization
* Business Intelligence & Analytics

## Insights Generated

The analysis highlights the rapid growth of AI-related careers worldwide and identifies the most sought-after roles such as Data Scientist, Machine Learning Engineer, and AI Researcher. It also reveals salary variations, skill demand trends, and the increasing adoption of remote AI jobs.

## Conclusion

This project demonstrates how Power BI can be used to transform raw job market data into meaningful insights that support career planning, recruitment strategies, and data-driven business decisions.
